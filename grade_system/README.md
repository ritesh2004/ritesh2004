# Python Grade System
## Description:-
#### This package will creat excel file automatically from user input. First of all when you will run this file it prompts user to input some required data like- Name,Roll No,Department,Sex,marks. Then it will make grade depending upon marks and make an excel file containg all data(which are mentioned above). It will save your time and increase productivity.
